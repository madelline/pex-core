# pex-core
PEX Core is an experiment in building a persistent execution layer for humans and AI.

The idea:
AI becomes significantly more useful when it understands context, memory, constraints, goals, and operating patterns.

This project explores:
- Context persistence
- Memory retrieval
- Execution planning
- Decision support
- Daily operating systems

Current MVP:
- Markdown-based memory system
- Daily execution planner
- Claude API integration
- Context loading engine

Long-term direction:
A stateful execution system that acts as an operating layer between intention and action.
